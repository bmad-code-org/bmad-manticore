# Onboarding a new creator

Load this file whenever the studio pulse check comes back "no studio yet", or the creator is clearly new to Manticore. The goal of the first session: the creator leaves knowing what the studio is, with setup done or scheduled, and with a concrete first project in mind.

## The one path

mc-setup's full interview is the only setup path: it installs the BMad core, then interviews the creator (identity and paths, editor, render consent, video style, CTAs, audio lanes, brand and voice, headshots, tools). Manny routes there and never improvises a partial setup. Set the expectation honestly: done well it takes 30 to 60 minutes, and it is the highest-leverage hour in the studio because every skill reads what it writes.

## What Manny explains while walking them in

In this order, conversationally, not as a lecture:

1. The shape of the thing. Ideas go in one end, a cut video with graphics and packaging comes out the other, and the creator approves everything that matters at four hard gates (outline, cutplan, beats, final). Their voice is the product; the pipeline orchestrates it and never overwrites it.
2. Render-first, the default they will confirm at setup: every cut iteration produces a watchable preview, the final render is offered at the end, and the editor timeline export plus all assets are always created as an exit ramp into Resolve, Premiere, or whatever they use. They are never locked in.
3. Interview-recording mode, the trick worth knowing on day one: the braindump conversation can happen with the camera already rolling. They read each question aloud prefixed with the marker cue, and the cut stage segments the recording mechanically, so the braindump becomes usable footage instead of just notes.
4. The style interview matters. It asks for creators they want to emulate (video links welcome) and echoes back the takeaways; density, image types, overlay taste, and CTAs all seed the Production Bible, which is why their first video will not look like sparse text cards.
5. What exists when setup finishes: a config every skill reads, a brand folder with bibles that learn, and an honest runnability report of what is ready and what is pending. The pending list is their highest-value next work, not a failure.

## First project advice

Suggest talking-head as the first format: shortest path through every stage, and the format the pipeline has the most mileage on. A creator arriving with existing footage skips all of this and goes straight to the footage-first flow (`references/flows.md`); setup still has to exist first.

## Automation questions

bmad-autopilot (ships in the bmad-bmm module) is an optional integration for scheduled or automated runs. Mention it only when a creator asks about automation; nothing in Manticore depends on it.
